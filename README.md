# Fast Video

[English](README_EN.md) | 简体中文

一个面向 Edge / Chromium 浏览器的视频倍速扩展。它修复了普通倍速插件持续覆盖网页播放器速度的问题，并把高倍速和自定义倍速直接加入哔哩哔哩播放器的倍速菜单。

## 功能

- 在 B 站播放器倍速菜单中提供 `0.5×` 至 `16×` 的常用速度。
- 支持输入 `0.25×–16×` 的自定义倍速，最多两位小数。
- 普通视频、番剧、影视和课程均可使用，全屏状态下也能调整。
- 当前视频切换分 P、暂停或恢复播放时保留所选速度。
- 打开新视频或刷新页面后默认恢复 `1×`。
- B 站始终由播放器菜单控制，不会被扩展强制锁定。
- 其他网站默认只设置一次；只有主动选择“锁定当前速度”后才会持续固定。
- 荷塘雨课堂兼容模式可稳定使用至 `3.5×`。

> 直播没有包含在支持范围内。直播流通常会自动追赶实时进度，高倍速行为并不稳定。

## 安装

本项目暂未发布到扩展商店，需要以解压缩扩展的方式安装。

1. 下载本仓库：点击 GitHub 页面右上方的 **Code → Download ZIP**。
2. 解压 ZIP 文件。
3. 在 Edge 地址栏打开 `edge://extensions`。
4. 打开页面左侧的“开发人员模式”。
5. 点击“加载解压缩的扩展”。
6. 选择包含 `manifest.json` 的项目目录。
7. 刷新已经打开的视频页面。

如果安装过其他倍速扩展，请先将它停用，避免多个扩展同时修改播放速度。

## 使用方法

### 哔哩哔哩

打开普通视频、番剧、影视或课程页面，将鼠标移到播放器的“倍速”按钮上。扩展后的菜单包含以下预设值：

```text
0.5×  0.75×  1×  1.25×  1.5×  2×
2.5×  3×     4×  5×     8×    16×
```

点击任意预设值即可切换。需要其他速度时，在菜单底部的“自定义”输入框中填写数值，然后按 Enter 或点击“应用”。

### 其他视频网站

1. 点击浏览器工具栏中的扩展图标。
2. 选择预设速度，或输入自定义速度后点击“应用”。
3. 此操作默认只设置一次，网页自身的倍速按钮仍可继续调整。
4. 如果确实需要固定某个网站的速度，点击“锁定当前速度”；再次点击即可解除。

### 荷塘雨课堂

Fast Video 会阻止雨课堂播放器把已选择的倍速强制改回 `1×`，并将设置同步到标签页内的播放器 iframe。

当前限制：

- 雨课堂最高稳定倍速为 `3.5×`。
- `4×` 及以上尚未支持，扩展弹窗会禁用对应选项。
- 暂未替换雨课堂播放器原有的倍速按钮；请使用浏览器工具栏中的扩展弹窗调速。
- 更新扩展后必须完整刷新课程页面，旧页面会出现 `Extension context invalidated`，无法继续使用新脚本。

## 更新本地版本

更新项目文件后，需要手动重新加载扩展：

1. 打开 `edge://extensions`。
2. 找到“倍速播放控制器・B站改良版”。
3. 点击“重新加载”。
4. 刷新视频页面。

## 常见问题

### 点击倍速后没有变化

- 确认旧的倍速扩展已经停用。
- 在 `edge://extensions` 中点击本扩展的“重新加载”。
- 刷新视频页面后重新尝试。
- 确认加载的是包含 `manifest.json` 的目录，而不是它的上级目录。

### 移动项目目录后扩展失效

Edge 会记录解压缩扩展的原始路径。请删除失效的扩展条目，再从新目录重新“加载解压缩的扩展”。

### 为什么需要访问所有网站？

扩展需要在含有视频的网页中查找媒体元素并设置播放速度。倍速设置保存在浏览器的扩展存储中；项目不包含统计、广告或向外部服务器上传数据的代码。

## 开发与测试

需要 Node.js 以及 Playwright：

```bash
npm install
npx playwright install chromium
npm test
```

测试覆盖 B 站预设倍速、自定义倍速、播放器回写旧速度、原生倍速控制，以及普通网站的显式锁定行为。
